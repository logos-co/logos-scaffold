# Program Deployment Scaffold (Local v0.1)

This project was generated by `logos-scaffold` for a local-only vertical slice.

## Scope

- Local sequencer only (`http://127.0.0.1:3040`)
- Deterministic wallet bootstrap
- Native-token topup + hello-world deploy/interact/verify
- DevNet is deferred for this version

## Quick Start

```bash
logos-scaffold setup
logos-scaffold localnet start
logos-scaffold wallet init
logos-scaffold slice run
```

For reliability checks (same slice three times in a row):

```bash
logos-scaffold slice run --repeat 3
```

## Explicit Vertical Slice

```bash
# Use account from `logos-scaffold wallet init` output, or create one with wallet directly.
ACCOUNT_ID="Public/<account-id>"

logos-scaffold wallet topup --to "$ACCOUNT_ID"
logos-scaffold deploy hello-world
logos-scaffold interact hello-world --account-id "$ACCOUNT_ID"
logos-scaffold verify hello-world --account-id "$ACCOUNT_ID"
```

## Useful Runtime Commands

```bash
logos-scaffold doctor
logos-scaffold localnet status
logos-scaffold localnet logs --tail 200
logos-scaffold localnet stop
logos-scaffold localnet reset
```

## Artifacts

Scaffold writes state/debug artifacts under `.scaffold/state/`:

- `wallet.state`
- `deploy.state`
- `slice-last.json`
