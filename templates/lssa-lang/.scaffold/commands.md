# Command References

- scaffold setup: `logos-scaffold setup`
- scaffold build + idl + client: `logos-scaffold build`
- idl only: `logos-scaffold idl build`
- client only: `logos-scaffold client build`
- standalone sequencer: `RUST_LOG=info target/release/sequencer_runner sequencer_runner/configs/debug`
- wallet home env: `export NSSA_WALLET_HOME_DIR=$(pwd)/.scaffold/wallet`
