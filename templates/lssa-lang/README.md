# LSSA Lang Scaffold

This project was generated by `logos-scaffold` with the `lssa-lang` template.

It keeps the existing program-deployment workflow and adds an Anchor-inspired layer:
- annotation-driven instruction/account metadata
- deterministic IDL generation
- generated Rust client integration with `wallet::WalletCore`

## First Run

```bash
logos-scaffold setup
logos-scaffold localnet start
export NSSA_WALLET_HOME_DIR=$(pwd)/.scaffold/wallet
logos-scaffold build
```

`logos-scaffold build` runs:
1. `logos-scaffold setup`
2. `cargo build --workspace`
3. `logos-scaffold idl build`
4. `logos-scaffold client build`

## Framework Commands

```bash
logos-scaffold idl build
logos-scaffold client build
logos-scaffold migrate --to lssa-lang --dry-run
```

## Token Pilot Example

1. Compile programs and generated clients:

```bash
logos-scaffold build
```

2. Create accounts (public/public/private):

```bash
wallet account new public
wallet account new public
wallet account new private
```

3. Run initialize and transfers through generated client runner:

```bash
cargo run --bin run_token_pilot -- init --to <public_account_id>
cargo run --bin run_token_pilot -- transfer --from <public_sender_id> --to <public_recipient_id> --amount 10
cargo run --bin run_token_pilot -- transfer --from <public_sender_id> --to Private/<private_recipient_id> --amount 7
```

4. Sync private state when needed:

```bash
wallet account sync-private
```
